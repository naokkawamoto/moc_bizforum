作成日: 2026年2月17日

更新（2026年4月8日）: ルート index.html の SYSTEM USER 導線（user_login.html 経由）、管理画面ログイン、共通ヘッダー・フッター、user_settings_staff.html、user_activity_log.html、user_visitor_detail.html の参加・不参加表示等を現行モックに合わせて反映。

更新（2026年4月9日）: 共通ヘッダー user_header.html のログインユーザー表示はナビバー右の1箇所のみとし、オフキャンバスメニュー内の人アイコン・氏名の重複表示を廃止（イベントリスト等すべてのページで同様）。

更新（2026年4月9日・サイト反映）: イベント配下画面の黒帯にイベント番号（例：NBCS2026-M0305）をイベント名の右に表示。イベント情報登録はイベント番号必須・参加規模・営業担当者登録（CSVボタン）・プログラムから開催日列削除・来場者リストから営業担当者列削除・備考削除。イベントトップはイベント概要（参加規模・備考なし）・営業担当者カード。来場・入退場管理はCSVのみ（PDFなし）・列定義は実装参照。来場者・名札管理は来場者表に役職列。入場者詳細はサイドメニューなし・イベント識別番号・役職・参加不参加は行ごとに編集／保存。メール管理タブを目次に追記。

対象デバイス: デスクトップ・タブレット（表示幅 1024px 以上を想定。例：iPad Pro 1024×1366px）

対応ブラウザ: Google Chrome、Safari、Microsoft Edge の各ブラウザについて、リリースから1年以内のバージョンを対象とする（例：2026年2月時点では各ブラウザの直近1年分のメジャーバージョン）。

注意: 受付アプリ（スマートフォン向け）の要件定義は user_site/requirements_specification.html を参照してください。
