CSVを追加した場合、重複を排除する。

抽選結果は当選・抽選漏れ・ブランクの値をそのまま取り込む（抽選漏れも除外しない）。

  • 出欠状況の値は、本人出席、代理出席、当日出席、交代出席、欠席、申込者、キャンセル、抽選漏れ。
  • メール配信は、出欠がキャンセル・申込者・抽選漏れの来場者には送らない。
  • 後からキャンセルになった人は手動でキャンセルにステータス変更したい。
  • 抽選結果の値は当選・抽選漏れ・ブランク（CSVでは上記どおり取り込み）。
