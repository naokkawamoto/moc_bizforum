    来場者 新規登録

    イベント登録画面の来場者リストと同じ項目です。QRコードは登録後に発行されます。

    空のときは「姓」「名」を結合して保存します。
    担当者絞り込みは山田・佐藤・鈴木の区分に合わせて集計されます。
    印刷設定
    用紙サイズ：はがきサイズ（100×148mm）で固定です。変更はできません。
    テンプレートを選ぶとレイアウト・配色などのデザインを変更できます。

    差し込み印刷要素

    名札に差し込める項目です。

    • 管理番号（イベント識別番号）
    • 会社名
    • 部署
    • 役職
    • お名前
    • 開催日
    • 開催場所
    • 来場受付コード（QRコード）
    全枚共通で印字する文言を変更できます。
    ロゴ画像の配置を変更できます。

    名札の印刷設定ですが、テンプレも確かに3パターン程用意しておきたいのですが、
    添付のようにデザインがある場合があります。
    こういう場合はデザインのPNGだったりPDFなどを差し込みしたい

    来場者リスト CSVアップロード

    SalesforceのCSVリストより、以下の項目を読み込みます。

    • イベント識別番号
    • 抽選結果
    • 出欠状況
    • 会社名
    • 部門
    • 役職
    • 氏名
    • 姓（カナ）
    • 名（カナ）
    • メールアドレス
    • 選択セッション
    • QRコード

    ※ 営業担当者は後で追加もできる。

    CSVを追加した場合、重複を排除する。

    抽選結果列の当選だけを抽出する。

    • 一括追加CSVするとき、抽選結果が当選、ブランク、キャンセル、抽選漏れ、申込者、代理出席のうち、抽選漏れは弾き、かつ、キャンセルと申込者はキャンセル表示されるようにしたい。
    • 後からキャンセルになった人は手動でキャンセルにステータス変更したい。
    • ※キャンセルの人にはメールは送られない。
