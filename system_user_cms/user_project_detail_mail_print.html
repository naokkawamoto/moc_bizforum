    来場者 新規登録

    イベント登録画面の来場者リストと同じ項目です。QRコードは登録後に発行されます。

    印刷設定
    用紙サイズ：はがきサイズ（100×148mm）で固定です。変更はできません。
    テンプレートを選ぶとレイアウト・配色などのデザインを変更できます。

    名札テンプレート（1イベント・複数）

    1つのイベントに対して複数の名札テンプレートを保持できます。下の帯（タブ）でテンプレを切り替え、表示・印刷に使うデザインを選びます（モック表示）。

    1イベントあたり名札テンプレートは5個まで保持できる想定です（要確認）。詳細は 名札テンプレート.pdf を参照してください。

    差し込み印刷要素

    名札に差し込める項目です。

    • 管理番号（イベント識別番号）
    • 会社名
    • 部署
    • 役職
    • 姓＋名（姓のみでも可。名は任意）
    • 選択セッション（複数ある場合は段落（改段落）で区切って差し込み。例：1行目「セッションA」／空行／次行「セッションB」…）
    • 開催日
    • 開催場所
    • 来場受付コード（QRコード）
    全枚共通で印字する文言を変更できます。
    ロゴ画像の配置を変更できます。

    名札の印刷設定ですが、テンプレも確かに3パターン程用意しておきたいのですが、
    添付のようにデザインがある場合があります。
    こういう場合はデザインのPNGだったりPDFなどを差し込みしたい

    来場者リスト CSVアップロード

    SalesforceのCSVリストより、以下の項目を読み込みます。

    • イベント識別番号
    • 抽選結果
    • 出欠状況
    • 会社名
    • 部門
    • 役職
    • 氏名
    • 姓（カナ）
    • 名（カナ）
    • メールアドレス
    • 選択セッション
    • QRコード

    ※ 営業担当者は後で追加もできる。

    CSVを追加した場合、重複を排除する。

    抽選結果は当選・抽選漏れを取り込みます。セルが空のときは空欄として取り込みます（CSVの空セルも空欄）。抽選漏れは除外しません。

    出欠状況は本人出席、代理出席、当日出席、交代出席、欠席、申込者、キャンセル、抽選漏れのいずれか、または空欄（セル未入力・CSVの空セル）です。

    • 編集画面のプルダウンでは先頭に－があり、－を選ぶと抽選結果・出欠状況とも空欄として保存されます。
    • 出欠状況の値は、本人出席、代理出席、当日出席、交代出席、欠席、申込者、キャンセル、抽選漏れ（または空欄）。
    • メール配信は、出欠がキャンセル・申込者・抽選漏れの来場者には送らない。
    • 後からキャンセルになった人は手動でキャンセルにステータス変更したい。
    • 抽選結果・出欠状況は、CSVでも上記のとおり取り込みます（空欄は空セルと対応）。
